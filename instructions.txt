# Projekt-Spezifikation — Asian Superstore Website (React + Vite + Neon + Netlify Functions)

## Kurzbeschreibung
Erstelle eine moderne, responsive und elegante Informations-Website für den Supermarkt **„Asian Superstore“**.  
Ziel: Präsentation von Marke, Produktkategorien, wöchentlichen Angeboten und Filialen.  

- Kein Online-Shop
- Deployment auf **Netlify**
- Admin-Dashboard mit globaler Speicherung über **Neon (Postgres)** + **Netlify Functions**
- Sprache: Deutsch
- Ton: freundlich, professionell, frisch, multikulturell
- Slogan: **„Bringt die Welt zusammen.“**

## 3. Daten & DB-Setup (Neon / Postgres)

### Tabellen
**categories**
- id (UUID PK)  
- name (Text)  
- slug (Text)  
- short_description (Text)  
- image (Text / URL)  
- icon (Text / URL)

**offers**
- id (UUID PK)  
- title (Text)  
- description (Text)  
- price (Decimal)  
- old_price (Decimal)  
- valid_until (Date)  
- image (Text / URL)  
- category_id (FK → categories.id)

Optional: `locations` für Filialen, `users` für Admins

### API / Serverless Functions
- `/netlify/functions/getCategories` → GET  
- `/netlify/functions/getOffers` → GET  
- `/netlify/functions/createCategory` → POST  
- `/netlify/functions/updateCategory` → PUT  
- `/netlify/functions/deleteCategory` → DELETE  
- `/netlify/functions/createOffer` → POST  
- `/netlify/functions/updateOffer` → PUT  
- `/netlify/functions/deleteOffer` → DELETE  

- Serverless Functions prüfen Auth via Netlify Identity / JWT  
- Frontend nutzt `fetch()` oder React Query / SWR für Daten
